# User Dashboard

A production-style user dashboard built with Next.js, TypeScript, and Tailwind CSS.

## Features
- Fetch users from JSONPlaceholder API
- Display users in a table
- Search users by name or email (debounced)
- View user details and posts
- Favourite users and sort them to top
- Persist favourites using localStorage

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Setup

```bash
npm install
npm run dev
